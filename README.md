# AI Resume Analyzer

AI Resume Analyzer is a web-based application developed using HTML, CSS, and JavaScript that helps job seekers evaluate and improve their resumes using Artificial Intelligence. The application allows users to upload their resume in PDF, DOCX, or TXT format and receive an instant analysis powered by Google's Gemini AI.

The system extracts text from the uploaded resume, analyzes its content, and generates detailed feedback including Resume Score, ATS (Applicant Tracking System) Score, Job Match Score, Skills Found, Missing Skills, Strengths, Weaknesses, and Personalized Improvement Suggestions.

Key Features:

* Upload and analyze resumes in multiple formats.
* AI-powered resume evaluation using Google Gemini API.
* ATS compatibility scoring.
* Identification of existing and missing skills.
* Detailed strengths and weaknesses analysis.
* Personalized suggestions for resume improvement.
* Modern and responsive user interface.
* Job description matching for targeted resume optimization.

Technologies Used:

* HTML5
* CSS3
* JavaScript (ES6)
* Google Gemini AI API
* PDF.js (for PDF text extraction)

Project Objective:
The main objective of this project is to help students, freshers, and professionals improve their resumes and increase their chances of passing ATS screening systems used by recruiters and companies.
